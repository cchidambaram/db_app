

Steps to Execute : 
===================

From command Prompt:
====================
1) Navigate to the folder of dbms.exe
2) Execute the program with the input script file.
   For Example
	c:\My Documents> dbms.exe script=xyz.txt > output.txt


Double Clicking the dbms.exe file:
===================================
1) When the dbms.exe file is double clicked ,input be provided .
2) For Example :
	Enter the Script file :
	script=xyz.txt > output.txt
 

Begin Transaction - commit /rollback will work only in the script file input.
